문단을 구분하는 태그 : p, pre

문단을 나누는 태그로 p태그와 pre 태그가 있다. p태그 안에서는 줄바꿈 처리를 하고자 한다면 별도의 태그를 사용해야함
**줄바꿈 처리**
그리고 공백은 한개의 공뱍만 표시 하므로 별도의 기호문구를 사용해야 한다
-->기호문구를 사용하지 않은 경우 : @ @
-->기호문구를 사용한 경우 : @     @

    pre 태그는 시작태그에서부터 종료태그까지 입력한 내용을

그 외의 글자 관련 태그들

일반적인 글자 표현하기 : 태그 없이도 글자 표시 가능

strong :글자를 굴게 표시하는 태그(+스크린 리더)

em : 글자를 기울여서 표시하는 태그

mark: 형광펜 효과와 함께 표시하는 태그

글자에 밑줄을 표시하는 태그

s: 글자에 취소선을 표시하는 태그

small : 글자를 작게 표시하는 태그

첨자 표시 태그
글자에 아래첨자를 표시하는 태그와 위첨자를 표시하는 태그
ex)a2 + b2 = c2
